Простое REST API приложение для работы с **пользователями и их
постами**, созданное на **FastAPI + SQLAlchemy + SQLite**.

## 📌 Возможности

-   Создание пользователей
-   Получение списка пользователей
-   Поиск пользователя по имени
-   Создание постов
-   Получение списка постов
-   Связь пользователя и постов
-   Автоматическая документация API

------------------------------------------------------------------------

# 🛠 Технологии

  Технология   Описание
  ------------ --------------------------------------
  FastAPI      Современный Python framework для API
  SQLAlchemy   ORM для работы с базой данных
  SQLite       Легкая встроенная база данных
  Pydantic     Валидация данных
  Uvicorn      ASGI сервер

# ⚙️ Установка

``` bash
git clone https://github.com/your-username/fastapi-users-posts.git
cd fastapi-users-posts
pip install fastapi uvicorn sqlalchemy
```

# 📡 API Примеры

## Создать пользователя

POST /users/

``` json
{
  "name": "Alex",
  "age": 25,
  "gender": "male"
}
```

## Получить пользователей

GET /users/

## Получить пользователя

GET /users/{name}

## Создать пост

POST /posts/

``` json
{
  "title": "My first post",
  "body": "Hello world",
  "author_id": 1
}
```

## Получить посты

GET /posts/
