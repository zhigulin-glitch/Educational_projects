Описание проекта

Это Telegram-бот с текстовой RPG игрой.
Игрок проходит сюжетное вступление, выбирает расу, получает предметы, исследует локации, управляет инвентарём и может сражаться на арене.

Игра построена на интерактивных кнопках Telegram и базе данных для хранения игроков и предметов.
